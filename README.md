# PocketBase
Backend-as-a-service in one file. Embedded SQLite, auth, file storage, and admin UI.
## Local Development

    docker compose up

API: http://localhost:8090 | Admin: http://localhost:8090/_/
## Hooks
Custom JS hooks go in pb_hooks/. See pb_hooks/main.pb.js.
## Deploy on StackBlaze
This template includes a stackblaze.yaml for deployment.
