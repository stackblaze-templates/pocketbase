# PocketBase

[![Maintained by StackBlaze](https://img.shields.io/badge/maintained%20by-StackBlaze-blue)](https://stackblaze.com)
[![Weekly Updates](https://img.shields.io/badge/updates-weekly-green)](https://github.com/stackblaze-templates/pocketbase/actions)
[![License](https://img.shields.io/github/license/stackblaze-templates/pocketbase)](LICENSE)

Backend-as-a-service in a single file. Embedded SQLite, real-time subscriptions, authentication, and file storage — all in one Go binary.

> **Credits**: Built on [PocketBase](https://pocketbase.io) by [Gani Georgiev](https://github.com/pocketbase). All trademarks belong to their respective owners.

## Deploy on StackBlaze

[![Deploy on StackBlaze](https://img.shields.io/badge/Deploy%20on-StackBlaze-orange)](https://stackblaze.com)

This template includes a `stackblaze.yaml` for one-click deployment on [StackBlaze](https://stackblaze.com).
